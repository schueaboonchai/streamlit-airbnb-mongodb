# 🏠 Airbnb Analytics Dashboard

Interactive Airbnb analytics dashboard built with Streamlit, MongoDB Atlas, Plotly, and AI-powered insights.

This project demonstrates dashboard development, cloud database integration, data visualization, and AI analysis using the MongoDB `sample_airbnb` dataset.

---

## 📊 Features

- Interactive Airbnb analytics dashboard
- KPI metrics and summary statistics
- Country, room type, and property filters
- Plotly visualizations and map analytics
- Airbnb listings preview
- Download filtered dataset as CSV
- AI-powered Airbnb advisor
- Executive-level AI insights
- Smart recommendation system
- AI chat assistant for Airbnb analysis

---

## 🧠 AI Features

This project includes AI functionality powered by:

- OpenRouter API
- DeepSeek V4 Flash (Free Model)

AI capabilities include:
- Executive business analysis
- Listing improvement recommendations
- Pricing strategy suggestions
- Customer targeting insights
- AI-generated Airbnb consulting
- Interactive AI chat assistant
